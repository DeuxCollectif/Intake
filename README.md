# Intake Form - BA Project Intake System

This repository contains the BA Intake Form project, built using GitHub Spec Kit for spec-driven development.

## Project Structure

- **spec.md** - Project specification and requirements
- **.github/** - GitHub configuration (issues, discussions, etc.)

## Getting Started

1. Read [spec.md](./spec.md) for the full project specification
2. Check [.github/](./.github/) for issue templates and project configuration

## Development Workflow

This project follows the Spec Kit methodology:
1. **Specify** - Define requirements in spec.md
2. **Plan** - Break down into tasks
3. **Tasks** - Create actionable work items
4. **Implement** - Build with AI agents or manually
